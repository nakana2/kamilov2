## 👨‍💻 О себе
Backend & DevOps инженер (5+ года).  
Делаю сервисы, которые спокойно живут в проде и не требуют шаманских танцев при каждом релизе.

### 🎯 Фокус
- **Backend:** REST/gRPC, интеграции, event-driven, фоновые воркеры
- **Performance:** кеширование, очереди, оптимизация БД, p95/p99
- **DevOps:** CI/CD, Docker/K8s, Terraform, окружения dev/stage/prod
- **Observability:** метрики, логи, трейсы, алерты, SLI/SLO, runbooks

---

## 🧩 Что считаю “хорошим продом”
- **Надежность:** timeouts, retries, rate-limit, circuit breaker, graceful shutdown, idempotency  
- **Данные:** индексы, планы запросов, миграции, транзакции, конкурентный доступ  
- **Архитектура:** контракты, версионирование API, DLQ, schema evolution  
- **Безопасность:** OAuth2/OIDC, RBAC, audit trail, секрет-менеджмент
