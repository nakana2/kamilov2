## 👨‍💻 О себе

Привет! Я **Jasurbek Kamilov** — Fullstack & DevOps инженер с 5+ годами опыта.

Делаю сервисы, которые спокойно живут в продакшене, масштабируются без боли и не требуют шаманских танцев при каждом релизе. Люблю чистый код, надёжную архитектуру и когда всё видно и контролируемо.

### 🎯 Фокус
- **Fullstack**: мощный бэкенд + современный фронтенд (React + Next.js)
- **Highload & Performance**: кэширование, очереди, оптимизация БД, p95/p99
- **DevOps**: CI/CD, Docker, Kubernetes, Infrastructure as Code
- **Observability**: метрики, логи, трейсы, SLO/SLI, алерты и runbooks

### 🧩 Что считаю «хорошим продом»
- Надёжность: timeouts, retries, rate limiting, circuit breaker, graceful shutdown, idempotency
- Данные: правильные индексы, миграции, транзакции, конкурентный доступ
- Архитектура: чёткие контракты, versioning API, DLQ, schema evolution
- Безопасность: OAuth2/OIDC, RBAC, audit trail, секрет-менеджмент
